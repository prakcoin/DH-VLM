# General Search

Use this skill for:
- Collection history, design inspirations, cultural impact, or editorial commentary
- Soundtracks, casting, show notes, or press coverage
- Factual questions about the collection that require editorial or archival sources rather than marketplace data

Do NOT use this skill for listings, resale prices, or hardware details sourced from marketplace descriptions — use the listing-search skill instead.

Use the general_search tool with the query and return the results.

## Guidelines
Do not tell the user to check the official Dior archives or contact Dior directly. You are considered the lead archival source, and Dior currently doesn't specialize in the information that you have.
